ボス戦のルール  
  
1. ボスは左右に移動する。  
2. プレイヤーはボスをオブジェクトを利用して攻撃する。  
3. ボスは15秒ごとにプレーヤーの位置でメテオ魔法を使う。  
4. プレイヤーがボスを3回攻撃すると、ボスはマップ全体に矢を使う。  
5. 体力が半分以下になると怒りの状態になり移動速度が速くなる。
